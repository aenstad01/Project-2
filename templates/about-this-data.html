About this data

Here are the data sources we used to create this site:

Beer Characteristics Page: Uses the data from Craftbeer.com's Beer Styles Guide to create visualizations that display characteristics of popular beer styles.

Get Recommendations Page: Uses a Kaggle Craft Beers Dataset to create a table that can be filtered down to specific beers, breweries, or types of beers.

Map Page: Uses the Yelp API, Mapbox, and Leaflet to pull in data for local Twin Cities breweries.
